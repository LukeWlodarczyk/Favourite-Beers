# Favourite Beers

Live: http://fav-beers.surge.sh/

## Technologies

* React
* React Router
* Redux
* Redux Thunk

### Installation

Install the dependencies

```sh
$ npm install
```
Run app

```sh
$ npm start
```

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).
